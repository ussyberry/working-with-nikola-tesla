# Minimal BOM — Driver (Module A) and Coil (Module B)

Module A — Driver PCB
- MCU/Controller: 1x, e.g., STM32F1xx series
- Isolated gate-driver IC: 1x
- Isolated DC-DC converter: 1x
- PLL component (optional): 1x
- Isolation interface parts (optocouplers/transformers): 1x–2x
- Gate-drive power conditioning components (decoupling caps, resistors): as needed
- EMI input filter components: 1x package
- Control interface header (J1): 1x
- Enclosure and mounting hardware: as needed

Module B — External Coil Assembly
- MOSFETs (Q1–Q4): 4x
- Heatsinks for MOSFETs: 2x
- Primary coil form: 1x
- Secondary coil and topload: 1x
- Insulation materials (sleeves, spacers, potting): as needed
- Coil interconnect cables/traces: as needed
- Temperature sensors (optional): 1x–2x
- Safety shielding/EMI shielding materials: as needed
